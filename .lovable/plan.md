

## Объединение страниц Услуги и Досуг

### Проблема

Обе страницы практически идентичны:
- Одинаковые активности (пляжи, реки, экскурсии, барбекю, морские прогулки)
- Одинаковые доп. услуги (трансфер, аренда, доставка)
- Одинаковая структура (hero + zigzag + extras + CTA)
- Одинаковые изображения

### Решение

Удалить страницу `/services`, объединив весь контент в `/leisure`. Страница Досуг уже имеет лучший дизайн (animated hero gallery). Навигацию обновить соответственно.

### Изменения

**1. Удалить `src/pages/Services.tsx`**

**2. Обновить `src/App.tsx`**
- Убрать import Services и маршрут `/services`
- Добавить redirect с `/services` на `/leisure` (для сохранения ссылок)

**3. Обновить `src/components/Navbar.tsx`**
- Убрать пункт «Услуги» из навигации
- Переименовать «Досуг» в «Досуг и услуги» (или оставить «Досуг»)

**4. Обновить `src/pages/Leisure.tsx`**
- Обновить title на «Досуг и услуги»
- Добавить категорию «Услуги» в слайдер категорий
- Перенести доп. услуги (трансфер, аренда, доставка) — уже есть, контент совпадает

Никакого нового контента добавлять не нужно — страница Leisure уже содержит все активности из Services плюс дополнительные (велопрогулки).



## План: добавить раздел «Блог» с 40 SEO-статьями

### Архитектура

Создаём 4 новых файла и модифицируем 3 существующих:

```text
src/
├── data/
│   └── blogPosts.ts          ← массив из 40 статей (slug, title, date, excerpt, content, image)
├── pages/
│   ├── Blog.tsx               ← список карточек статей с поиском
│   └── BlogPost.tsx           ← страница отдельной статьи
├── components/
│   └── Navbar.tsx             ← добавить «Блог» между «Досуг» и «Отзывы»
├── App.tsx                    ← маршруты /blog и /blog/:slug
└── public/
    └── sitemap.xml            ← добавить /blog
```

### 1. Данные статей (`src/data/blogPosts.ts`)

Массив из 40 объектов:
- `slug` — URL-путь (например `otdyh-v-tenginke-s-detmi`)
- `title` — заголовок статьи
- `date` — дата публикации
- `excerpt` — краткий анонс (1-2 предложения)
- `image` — URL фото с Unsplash (природа Краснодарского края, море, домики)
- `content` — полный текст статьи в виде строки с HTML-разметкой (H2/H3, параграфы, списки)

Первые 5 статей — с полным текстом (800-1200 слов каждая), остальные 35 — с заголовком и анонсом (content = excerpt).

### 2. Страница списка (`src/pages/Blog.tsx`)

- Navbar + Footer (как на Leisure)
- Заголовок «Блог» с Badge
- Сетка карточек (responsive: 1 / 2 / 3 колонки)
- Каждая карточка: превью-фото, дата, заголовок, анонс, кнопка «Читать»
- Пагинация: 9 статей на страницу (state-based, без URL)
- ScrollReveal анимация

### 3. Страница статьи (`src/pages/BlogPost.tsx`)

- Navbar + Footer
- Хлебные крошки: Главная → Блог → Название статьи
- Hero-изображение статьи
- Заголовок, дата
- Контент статьи (dangerouslySetInnerHTML для HTML-контента)
- CTA-блок внизу: «Забронировать домик» → ссылка на `/#booking`
- Блок «Читайте также» — 3 случайные статьи

### 4. Навигация и маршруты

- `Navbar.tsx`: добавить `{ name: "Блог", link: "/blog", id: "blog", isPage: true, icon: <BookOpen /> }` между «Досуг» и «Отзывы»
- `App.tsx`: добавить `<Route path="/blog" element={<Blog />} />` и `<Route path="/blog/:slug" element={<BlogPost />} />`
- `sitemap.xml`: добавить `/blog`

### 5. SEO каждой статьи

- `document.title` = заголовок статьи + «| База отдыха Сон»
- Meta description через `<meta>` tag в useEffect
